EcoChain AI

智能碳管理平台

项目概述

EcoChain AI是一个基于人工智能的碳足迹追踪与管理平台，帮助企业和个人精确计算、分析和减少碳排放。 平台支持多种数据源集成，包括API、数据库等，实现自动化数据采集和分析。

最新更新：添加了完整的后端集成、用户认证系统和多数据源导入功能。

主要功能

碳足迹追踪

记录和追踪各类活动产生的碳排放，支持手动输入和自动采集。

数据分析

可视化分析碳排放趋势和模式，生成专业报告。

AI减排策略

基于AI的减排建议和优化方案，帮助用户减少碳排放。

生态链管理

追踪供应链和价值链中的碳排放，实现全链条管理。

新增功能

用户认证系统

完整的注册、登录、密码重置功能，保障数据安全。

多数据源集成

支持API、PostgreSQL、MySQL、MongoDB、SQL Server、Snowflake等多种数据源。

定时数据同步

自动定期同步数据源数据，减少手动操作。

数据映射

自定义数据源字段到系统字段的映射，灵活适配不同数据格式。

如何启动

# 安装依赖
npm install

# 启动开发服务器
npm run start

# 或者分别启动
npm run dev       # 前端
npm run server    # 后端

启动后，访问 http://localhost:3000 查看应用
