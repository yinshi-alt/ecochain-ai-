# EcoChain AI - 智能碳管理平台

## 快速开始

### 环境要求
- Node.js 14+
- npm 6+

### 安装步骤

1. **克隆代码库**
```bash
git clone https://github.com/yourusername/ecochain-ai.git
cd ecochain-ai
```

2. **安装依赖**
```bash
npm install
```

3. **启动应用**
```bash
npm run start
```

4. **访问应用**
- 前端: http://localhost:3000
- 后端API: http://localhost:5000
